# HybridMind CLI Agent

Usage: `python cli/agent.py [options]`

## Commands
- `/memory` - Show last recalled memories
- `/stats` - Call memory.stats() and pretty-print info
- `/sessions` - List all sessions
- `/archive` - Archive current session and exit
- `/forget <text>` - Recall closest node to text and forget it
- `/clear` - Clear terminal
- `/help` - Show command list
- `/exit` or `/quit` - Exit cleanly
